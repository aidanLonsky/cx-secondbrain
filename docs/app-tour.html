A visual tour of secondbrain

Your tiny, terminal-powered idea catcher.

secondbrain turns a passing thought into a Markdown note, then helps you find it again. No database, no accounts—just files you own.

Start here

Ask it what it can do.

The built-in --help flag reveals one option and three friendly commands. This is the app’s actual command-line menu.

your terminal
$ secondbrain --help

Usage: secondbrain [OPTIONS] COMMAND [ARGS]...

  Capture and manage Markdown notes.

Options:
  --help  Show this message and exit.

Commands:
  list  List Markdown notes in the configured directory.
  new   Create a Markdown note containing TEXT.
  show  Print the note identified by its list NUMBER.
✍️

1. Catch a thought

Use new TEXT to place exactly what you type into a fresh Markdown note.

secondbrain new "Try that café"needs: TEXT
🗂️

2. Browse your notes

Use list to see every lowercase .md note, ordered and numbered.

secondbrain listneeds: nothing
🔎

3. Read one back

Use show NUMBER with a number from the latest list to print that note in your terminal.

secondbrain show 2needs: NUMBER
